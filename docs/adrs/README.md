# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting significant technical decisions for gcm.

## Quick Reference

| Status | Meaning |
|--------|---------|
| **Proposed** | Draft, pending review |
| **RFC** | Request for Comments - seeking team input |
| **Accepted** | Approved and immutable |
| **Superseded** | Replaced by newer ADR |
| **Rejected** | Considered but not adopted |

## Creating a New ADR

Use the `/adr` command or `/adr CLO-XX` to create from a Linear task.
